## Overview

The `Sidebar` component is the primary UI for customizing uniforms. It provides a sticky sidebar with collapsible sections for fabric types, designs, colors, text fields, logo upload, and player management.

**Location:** `src/components/Sidebar.jsx`

## Component Structure

```jsx
export default function Sidebar() {
  // State management
  // Event handlers
  // Render collapsible sections
}
```

The `Sidebar` component does not accept props. It manages its own state and uses React Router hooks.

## Key Functions

### handleSave

Navigates to the design summary page with all design data and players.

```jsx
const handleSave = () => {
  const designData = { 
    fabricType, 
    design, 
    colors, 
    teamName, 
    playerName: playerNameField, 
    logoUrl, 
    teamFont, 
    playerFont 
  };
  navigate("/datos-seleccionados", { 
    state: { design: designData, players, modelo } 
  });
};
```

**Navigation State:**
- `design` - All customization options
- `players` - Player roster array
- `modelo` - Route parameter for shirt model

### State Restoration

Restores state when navigating back from summary page:

```jsx
useEffect(() => {
  if (locationState?.design) {
    const design = locationState.design;
    setFabricType(design.fabricType || "");
    setDesign(design.design || "");
    setColors(design.colors || { torso: "", back: "", sleeves: "", collar: "" });
    setTeamName(design.teamName || "");
    setPlayerNameField(design.playerName || "");
    setLogoUrl(design.logoUrl || "");
    setTeamFont(design.teamFont || "");
    setPlayerFont(design.playerFont || "");
  }
  
  if (locationState?.players) {
    setPlayers(locationState.players);
  }
}, [locationState]);
```

## Section Component

Collapsible section wrapper used throughout the sidebar.

### Props

<ParamField path="title" type="string" required>
  Section header text
</ParamField>

<ParamField path="icon" type="Component" required>
  Heroicon component to display in header
</ParamField>

<ParamField path="children" type="ReactNode">
  Content to display when expanded
</ParamField>

### Implementation

```jsx
function Section({ title, icon: Icon, children }) {
  const [open, setOpen] = useState(true);
  const id = title.replace(/\s+/g, "-").toLowerCase();

  return (
    <section>
      <header>
        <button
          type="button"
          onClick={() => setOpen(!open)}
          aria-expanded={open}
          aria-controls={id}
          style={{ fontFamily: "Arboria" }}
          className="w-full flex justify-between items-center text-left
                     transition bg-personalizado px-3 py-2 rounded-lg"
        >
          <span className="flex items-center gap-2">
            {Icon && <Icon className="w-5 h-5 text-purple-700" />}
            {title}
          </span>
          <ChevronDownIcon
            className={`w-5 h-5 transition-transform duration-200 
                       ${open ? "rotate-180" : ""}`}
          />
        </button>
      </header>
      {open && children && (
        <div id={id} className="mt-3 grid grid-cols-2 gap-2">
          {children}
        </div>
      )}
    </section>
  );
}
```

## PlayersSection Component

Manages player roster with add, remove, and edit capabilities.

### Props

<ParamField path="players" type="Player[]" required>
  Array of player objects
</ParamField>

<ParamField path="setPlayers" type="(players: Player[]) => void" required>
  State setter function from parent
</ParamField>

### Internal Functions

#### handleAdd

Creates new player with empty fields:

```jsx
const handleAdd = () => {
  const newPlayer = {
    id: Date.now(),
    name: "",
    gender: "",
    jersey: "",
    number: "",
    quantity: 1,
  };
  setPlayers([...players, newPlayer]);
};
```

#### handleRemove

Filters out player by ID:

```jsx
const handleRemove = (id) => {
  setPlayers(players.filter((p) => p.id !== id));
};
```

#### handleChange

Updates single field for specific player:

```jsx
const handleChange = (id, field, value) => {
  setPlayers(
    players.map((p) => (p.id === id ? { ...p, [field]: value } : p))
  );
};
```

### UI Features

- **Eye icon button**: Preview player's uniform (currently non-functional)
- **Gender dropdown**: H (Hombre), M (Mujer), Ñ (Niño)
- **Jersey size dropdown**: S, M, L
- **Number input**: Jersey number
- **Quantity input**: Number of jerseys to order
- **Delete button**: Red X button to remove player
- **Selection highlight**: Purple ring when player is clicked

<Warning>
**Event Propagation**: The component uses `e.stopPropagation()` to prevent row selection when interacting with inputs.
</Warning>
