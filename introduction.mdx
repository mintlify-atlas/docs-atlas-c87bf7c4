## What is Dark Lion Custom Uniforms?

Dark Lion Custom Uniforms is an interactive web application that allows esports teams and gaming organizations to design custom jerseys and apparel with real-time 3D visualization. Built for the Dark Lion brand, this platform streamlines the uniform ordering process from design to delivery.

## Built With Modern Technology

Dark Lion Custom Uniforms leverages cutting-edge web technologies to deliver a smooth, interactive experience:

- **React 19** — Modern UI framework with latest features
- **React Three Fiber** — 3D graphics rendering with Three.js
- **Vite** — Lightning-fast build tool with HMR
- **Tailwind CSS** — Utility-first styling with custom design tokens
- **React Router** — Seamless navigation and state management
