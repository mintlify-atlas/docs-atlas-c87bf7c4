## Overview

Dark Lion Custom Uniforms provides a comprehensive, interactive platform for designing esports uniforms online. This guide walks you through the complete workflow from selecting your base shirt style to submitting your final order.

<Note>
The entire customization process is done 100% online with real-time preview of your design choices.
</Note>

## Complete Customization Workflow

<Steps>
  <Step title="Select Your Shirt Style">
    Navigate to the home page where you'll see four available shirt styles:
    
    - **CAMISA CUELLO V** - V-neck style uniform
    - **CUELLO REDONDO** - Round neck style uniform
    - **CAMISOLA** - Jersey style uniform
    - **TANK TOP** - Sleeveless style uniform
    
    Click the "Personalizar" button on any shirt style to begin customization.
    
    **Navigation paths:**
    - V-Neck: `/personalizar/cuello-v`
    - Round Neck: `/personalizar/cuello-redondo`
    - Jersey: `/personalizar/cuello-camisola`
    - Tank Top: `/personalizar/cuello-tank-top`
    
    <Tip>
    Each shirt style displays with a watermark background and drop-shadow effects for better visualization.
    </Tip>
  </Step>

  <Step title="Choose Your Fabric Type">
    Once in the customization interface, start by selecting your fabric from the sidebar:
    
    - **Microfibra** - Lightweight microfiber material
    - **Poliester** - Durable polyester blend
    - **Nylon** - Flexible nylon fabric
    - **Dry-Fit** - Moisture-wicking dri-fit technology
    
    The fabric selector is located in the collapsible "Tipos de Tela" section at the top of the sidebar. Click any fabric option to select it - the selected option will be highlighted with a darker background.
    
    <Note>
    All fabrics feature dri-fit technology for optimal performance during gameplay.
    </Note>
  </Step>

  <Step title="Select Design Template">
    Choose from available design templates in the "Diseños" section:
    
    - Browse through pre-designed shirt templates
    - Click any design image to select it
    - Selected designs show a purple ring border (ring-2 ring-purple-500)
    - Design options include: `/shirt1.png`, `/shirt2.png`
    
    The design template provides the base visual structure for your uniform, which you'll then customize with colors and branding.
  </Step>

  <Step title="Customize Colors">
    This is where your design comes to life. Customize colors for four different shirt areas:
    
    ### Available Color Sections:
    
    **1. Cuerpo (Body/Torso)**
    - Main body color visible from the front
    - Located in "Cuerpo" section with Square3Stack3D icon
    
    **2. Espalda (Back)**
    - Back panel color
    - Located in "Espalda" section with ArrowUpTray icon
    
    **3. Mangas (Sleeves)**
    - Sleeve color selection
    - Located in "Mangas" section
    
    **4. Cuello (Collar)**
    - Collar/neckline color
    - Located in "Cuello" section
    
    ### Color Palette:
    
    You have access to **42 preset colors** organized in a 7-column grid:
    
    - **Row 1:** Grays and neutrals (white to black)
    - **Row 2:** Reds and burgundy tones
    - **Row 3:** Oranges and yellows
    - **Row 4:** Lime greens and teals
    - **Row 5:** Blues and navy shades
    - **Row 6:** Purples and pinks
    
    Each color button displays as an 8x8 circle and scales on hover (hover:scale-140) for easy selection. Selected colors show a 2px purple ring.
    
    <Tip>
    Click the section headers to collapse/expand color palettes. The chevron icon rotates 180° when expanded.
    </Tip>
  </Step>

  <Step title="Add Team Name and Branding">
    Personalize your uniform with team identity in the "Nombre del Equipo" section:
    
    ### Team Name:
    - Enter your team name in the text input field
    - Placeholder text: "Equipo..."
    - Input features purple focus border on selection
    
    ### Font Selection:
    Choose from 4 typography options:
    - **INTER** - Clean, modern sans-serif
    - **ACTOR** - Bold display font
    - **MONTSSERRAT** - Classic geometric sans-serif
    - **JOMHURIA** - Stylized display font
    
    Font buttons display in a 2-column grid with the actual font family applied to preview the style.
    
    <Note>
    The selected field shows a purple ring border for easy identification of which text element you're currently editing.
    </Note>
  </Step>

  <Step title="Add Player Name and Font">
    Customize individual player information in the "Nombre del Jugador" section:
    
    ### Player Name:
    - Enter player name in the text field
    - Placeholder: "Jugador..."
    - Same purple focus border as team name
    
    ### Font Selection:
    - Same 4 font options as team name
    - Fonts can be different between team and player text
    - Preview the font style directly on the button
    
    This creates the base player name that will appear on uniforms, which can be customized per player in the next step.
  </Step>

  <Step title="Upload Team Logo">
    Add your team's logo to the uniform design:
    
    **Location:** "Logo" section in sidebar
    
    **How to upload:**
    1. Click the large "UPLOAD" button (features ArrowUpTray icon)
    2. Opens logo upload interface at `/agregar-logo`
    3. Upload your logo file
    4. Logo URL is saved to design configuration
    
    The upload button spans the full width with purple background (bg-personalizado-click) and hover effects.
    
    <Warning>
    Logo files should be optimized for best performance. See the Best Practices guide for logo specifications.
    </Warning>
  </Step>

  <Step title="Add Player Details">
    Create individual player entries in the "Agregar Jugadores" section:
    
    ### For Each Player:
    
    **Quick Actions:**
    - **Eye Icon** - Preview button for viewing this player's uniform
    
    **Player Information:**
    - **Gen (Gender):** Select H (Hombre/Male), M (Mujer/Female), or Ñ (Niño/Child)
    - **Jersey:** Choose size - S, M, or L
    - **Num:** Player number (numeric input)
    - **Canti:** Quantity of uniforms for this player
    - **Name:** Full player name (text input)
    
    **Interface Details:**
    - All inputs feature purple backgrounds (bg-purple-200)
    - Selected player shows purple ring border
    - Red delete button (✕) to remove player
    - Green + button at bottom to add new players
    
    ### Adding Multiple Players:
    1. Click the green "+" button in the bottom-right
    2. Each new player gets a unique timestamp ID
    3. Fill in details for each player
    4. Players stack vertically with spacing
    
    <Tip>
    Start with one sample player (Arath B., #21, L, Quantity: 21) pre-populated as an example.
    </Tip>
  </Step>

  <Step title="Save Your Design">
    Once all customization is complete:
    
    1. Click the **"Guardar Uniforme"** button at the bottom of the sidebar
    2. Button features:
       - Full width (w-full)
       - Purple gradient background (bg-personalizado)
       - Scale animation on hover (hover:scale-110)
       - Montserrat1 font family
    
    **What Gets Saved:**
    - Fabric type selection
    - Design template choice
    - All color selections (torso, back, sleeves, collar)
    - Team name and font
    - Player name and font
    - Logo URL
    - Complete player list with details
    - Selected shirt model
    
    The system navigates to `/datos-seleccionados` with all design data passed via React Router state.
  </Step>

  <Step title="Review Your Order">
    On the order review page, you'll see:
    
    ### Left Sidebar - "MI UNIFORME PERSONALIZADO":
    
    **Configuration Details:**
    - Fabric type
    - Design template thumbnail (12x12 image)
    - Team name and font
    - Player name and font  
    - Logo thumbnail (12x12 image)
    
    **Color Swatches:**
    - Body color (6x6 color square)
    - Back color
    - Sleeves color
    - Collar color
    
    **Player List:**
    - Each player in purple card (bg-purple-100)
    - Shows name, gender, jersey size, number, quantity
    - Compact text format
    
    ### Center View:
    - Large uniform preview image
    - Two smaller thumbnails below (front and back views)
    - Thumbnails scale on hover (hover:scale-105)
    
    ### Action Buttons:
    - **"Editar Uniforme"** (Purple) - Returns to customization with all data preserved
    - **"Enviar Uniforme"** (Green) - Proceeds to contact form
    
    <Note>
    The edit function preserves all your design choices using React Router state, so you can make changes without losing work.
    </Note>
  </Step>

  <Step title="Submit Contact Information">
    Click "Enviar Uniforme" to reach the final step:
    
    ### Contact Form Fields:
    
    **Required Information:**
    - **Nombre** (Name): Minimum 3 characters
    - **Teléfono** (Phone): Exactly 10 digits
    - **Correo** (Email): Valid email format
    
    ### Validation:
    - Real-time error messages display below each field
    - Errors clear when you start typing
    - All fields validated before submission
    
    ### Submission Process:
    1. Fill all contact fields
    2. Click submit button
    3. System sends POST request to API endpoint
    4. Payload includes:
       - Contact information
       - Design file URL
       - Fabric selection
       - Aggregated sizes by type (CABALLERO/DAMA)
       - Complete player roster with details
    
    **API Endpoint:** `https://api-darklion.onrender.com/api/designs/client-and-design`
    
    ### After Submission:
    - Success alert confirmation
    - Form resets to blank state
    - Team contacts you to confirm details and begin production
    
    <Tip>
    Have all your player information ready before starting to streamline the process.
    </Tip>
  </Step>
</Steps>

## Visual Workflow Summary

The entire interface features:
- Watermark background throughout (MarcaAgua.png at 30% opacity)
- Sticky sidebar with backdrop blur effect
- Collapsible sections with chevron indicators
- Purple theme (morado-exito, morado-claro colors)
- Smooth transitions and hover effects
- Montserrat1 and Arboria font families

## Navigation Between Steps

- **Forward Progress:** "Guardar Uniforme" → "Enviar Uniforme" → Submit
- **Edit Functionality:** Review page → "Editar Uniforme" → Returns to customization with state
- **State Preservation:** All data passed via React Router location state

<Note>
If you navigate away without saving, you'll need to start over. Always click "Guardar Uniforme" to preserve your work.
</Note>
