## Overview

The Dark Lion customization engine provides a comprehensive design system that allows teams to create fully personalized esports uniforms. Users can customize every aspect of their uniform, from fabric type to individual player details, all within an intuitive sidebar interface.

<Note>
  All customization happens in real-time with the 3D viewer updating as you make changes. Your design is automatically saved when you click "Guardar Uniforme".
</Note>

## Customization Interface

The customization page (`src/pages/Customize.jsx`) consists of two main components:

- **Sidebar Panel**: Contains all customization controls
- **3D Viewer**: Real-time preview of your uniform design

## Saving Your Design

The "Guardar Uniforme" button at the bottom of the sidebar saves all customization data:

### Saved Data Structure

When you save, the following information is collected:

```javascript
{
  fabricType: "Selected fabric",
  design: "/shirt1.png or /shirt2.png",
  colors: {
    torso: "rgb(...)",
    back: "rgb(...)",
    sleeves: "rgb(...)",
    collar: "rgb(...)"
  },
  teamName: "Your team name",
  playerName: "Player name",
  logoUrl: "Logo URL",
  teamFont: "Selected font",
  playerFont: "Selected font"
}
```

**Navigation**: After saving, you're redirected to `/datos-seleccionados` with your design data (`src/components/Sidebar.jsx:53-56`).

### State Persistence

The system supports returning to the customization page:

- **Location State**: Design data is passed via React Router state
- **Auto-restore**: Previous selections are automatically loaded when returning from the review page
- **No Data Loss**: All customization work is preserved during navigation

**Code Reference**: `src/components/Sidebar.jsx:35-51`

## Best Practices

<Tip>
  **Design Workflow**:
  1. Start with fabric type and design template
  2. Choose your color scheme for all uniform areas
  3. Add team and player text with appropriate fonts
  4. Upload your team logo
  5. Add all team members with their details
  6. Save and review your design
</Tip>

<Note>
  The customization interface is fully responsive and optimized for desktop use. All changes are tracked in real-time, so you can experiment freely before saving.
</Note>

## Technical Implementation

The customization system is built with:

- **React Hooks**: `useState` for design state management
- **React Router**: `useNavigate`, `useParams`, `useLocation` for navigation
- **Heroicons**: Icon library for visual indicators
- **Tailwind CSS**: Utility-first styling
- **Component Architecture**: Modular design with reusable components

**Main Files**:
- `src/pages/Customize.jsx`: Main customization page
- `src/components/Sidebar.jsx`: Customization controls
- `src/components/ShirtViewer3D.jsx`: 3D preview component
